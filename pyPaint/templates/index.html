Python Paint App
A simple painting application made with Python, Django, and JavaScript/jQuery
Demo GitHub

This is a simple Web painting application built using the Django framework and JavaScript to allow interaction with the canvas.

Controls
  • Simple Brush
  • Eraser
  • Pattern Stamp Tool
  • Color Picker
  • Brush Size Slider
  • Clear Canvas Button
 Features
  • Resize Brush/Eraser/Stamp
  • Stores size for each tool
  • Stores color for each tool
  • User account creation
  • Save canvas to account
  • Account view
  • View user saved art
  • View recent activity
 Features to be added
  • Additional brush shapes
  • Additional stamp shapes
  • Line/Shape tool
  • Fill tool
  • Undo/Redo
  • Load/Paste Image
  • Delete saved art

Tech Stack:

  • Python
  • Django
  • Bootstrap
  • Canvas API
  • JavaScript / jQuery
  • Font Awesome and Bootstrap icons
  • Google Fonts
  • Native color picker and range inputs
  • Python Anywhere hosting
